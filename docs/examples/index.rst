.. Example scripts frontend

Example scripts
===============

This part of the documentation covers the example scripts provided with pysap.
Some examples require the optional dependencies installed with ``pysap[examples]``.
The ``diag_render_login_screen`` example also requires a manual ``wxPython``
installation, and ``router_password_check`` requires ``fau_timer`` from
``mona-timing-lib`` because those packages are not covered by
``requirements-examples.txt``.

For practical offline workflows, see the file format notebooks for
:doc:`SAPCAR archive inspection and extraction <../fileformats/SAPCAR>`,
:doc:`SAP Credv2 parsing <../fileformats/SAPCredv2>`,
:doc:`PSE parsing <../fileformats/SAPPSE>`, and
:doc:`SSFS key/data reads <../fileformats/SAPSSFS>`.
